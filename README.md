### 📱 Transform Your Business Communication
**NPM RCS** is a powerful, enterprise-grade Rich Communication Services (RCS) platform that enables businesses to send interactive, branded, and verified messages directly to customers' default messaging apps.

[**🌐 Visit Website**](https://rcs.npminfo.com) | [**📥 Download Client**](https://rcs.npminfo.com/download) | [**📖 API Docs**](https://rcs.npminfo.com/api-documentation)

---

## ✨ Key Features

- **💎 Rich Media Support**: Send high-quality images, videos, and carousels.
- **⚡ Interactive Buttons**: Add "Reply" or "Open URL" buttons inside messages.
- **🛡️ Secure Tunnel Client**: Connect local APIs to the cloud with end-to-end security.
- **✅ Verified Sender**: Build trust with verified business profiles and checkmarks.
- **📊 Real-time Analytics**: Track delivery, read receipts, and button clicks.
- **📦 Bulk Sending**: Robust queue processor for high-volume campaigns.

## 🚀 Getting Started

1. **Download**: Get the latest [RCS Setup MSI](https://rcs.npminfo.com/download).
2. **Install**: Run the installer and launch the RCS Tunnel Client.
3. **Authenticate**: Scan the QR code using your mobile device (Google Messages).
4. **Connect**: Use the provided API Key to start sending messages.

```bash
# Example API Request
curl -X POST https://api.rcs.npminfo.com/v1/send \
     -H "Authorization: Bearer YOUR_API_KEY" \
     -d "mobile=91XXXXXXXXXX" \
     -d "message=Hello from NPM RCS!"
```
